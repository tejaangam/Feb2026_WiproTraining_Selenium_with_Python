Here are medium to tough level MCQs on Constructor, Inheritance, and Polymorphism in Python (without answers).
Answer key is provided at the end.
 
1: Constructors in Python
1. What will be the output?
class A:
    def __init__(self):
        print("A Constructor")
class B(A):
    pass
obj = B() 
A) Error
B) A Constructor
C) Nothing
D) B Constructor
ANS:B

2. What happens if a class does not define __init__()? 
A) Compilation error
B) Default constructor is provided automatically
C) Object cannot be created
D) It inherits __init__ only if explicitly called
ANS:B

3. What is true about constructors in Python? 
A) Multiple constructors are allowed
B) Constructor name can be anything
C) Constructor is executed automatically when object is created
D) Constructor must return a value
ANS:A WRONG | ANS:C

4. What will be the output?
class Test:
    def __init__(self):
        print("Hello")
    def __init__(self):
        print("World")
t = Test() 
A) Hello
B) World
C) Hello World
D) Error
ANS: B
# PYTHON FOLLOWS LAST DEFINITION WINS 

5. If a parent class has a parameterized constructor, what must child class do? 
A) Nothing
B) Must override constructor
C) Must explicitly call parent constructor
D) Cannot inherit
ANS:C

2: Inheritance

6. What type of inheritance is this?
class A: pass
class B(A): pass
class C(B): pass
A) Multiple
B) Multilevel
C) Hierarchical
D) Hybrid
ANS:B

7. What will be the output?
class A:
    def show(self):
        print("A")
class B(A):
    def show(self):
        print("B")
obj = B()
obj.show()
A) A
B) B
C) Error
D) A B
ANS:B overridding

8. Which method determines method lookup order? 
A) dir()
B) mro()
C) help()
D) isinstance()
ANS:B

9. What is the diamond problem related to? 
A) Constructor overloading
B) Multiple inheritance
C) Method overriding
D) Encapsulation
ANS:B

10. What will be the output?
class A:
    def show(self):
        print("A")
class B(A):
    def show(self):
        print("B")
class C(A):
    def show(self):
        print("C")
class D(B, C):
    pass
obj = D()
obj.show()
A) A
B) B
C) C
D) Error
ANS:C WRONG | ANS: B{Because D(B, C) lists B first, it looks in B before C.}

3: Polymorphism
11. Polymorphism means:
A) Same function name with different behavior
B) Different function names
C) Multiple classes
D) Data hiding
ANS:A

12. What type of polymorphism is this?
print(len("Hello"))
print(len([1,2,3])) 
A) Method overriding
B) Operator overloading
C) Function overloading
D) Duck typing
ANS:D

13. What will be the output?
class Animal:
    def speak(self):
        print("Animal speaks")
class Dog(Animal):
    def speak(self):
        print("Dog barks")
a = Animal()
d = Dog()
a.speak()
d.speak()
A) Animal speaks
Animal speaks
B) Animal speaks
Dog barks
C) Dog barks
Animal speaks
D) Error
ANS:B

14. What is duck typing? 
A) Type checking using isinstance()
B) Behavior-based polymorphism
C) Constructor chaining
D) Operator overloading
ANS:B {it focuses on what an object can do, not what it is.}

15. What will be the output?
